## 📌 Project Overview
This project is the **Capstone Project** for the Meta Android Developer Professional Certificate. It represents a high-fidelity mobile solution for **Little Lemon Restaurant**, focusing on a seamless user experience (UX) and robust data persistence.

As a developer with a background in **Applied HR Management** and **Psychology**, this app was built with the principle of "Humanizing Digital Workspaces" in mind—ensuring that technical efficiency meets psychological comfort through intuitive navigation and reliable data handling.

## 🚀 Key Features
*   **Smart Onboarding:** A personalized entry point that captures user data and maintains state across sessions using `SharedPreferences`.
*   **Dynamic Home Screen:** Features a **Hero Section** with restaurant descriptions and a real-time search bar.
*   **Offline-First Data:** Integrated **Room Database** to store and display menu items fetched from a remote API, ensuring functionality even without internet.
*   **Interactive Menu Breakdown:** Advanced filtering system allowing users to sort food items by category (Starters, Mains, Desserts, etc.).
*   **Stack Navigation:** Implementation of a robust navigation graph that allows users to move fluidly between Home and Profile screens.

## 🛠 Tech Stack
*   **Language:** Kotlin
*   **UI Framework:** Jetpack Compose (Modern Declarative UI)
*   **Local Database:** Room Database (SQLite abstraction)
*   **Networking:** Ktor Client for asynchronous API calls.
*   **Image Loading:** Coil for optimized image caching.
*   **Architecture:** MVVM (Model-View-ViewModel) logic for separation of concerns.

## 📊 Strategic Implementation
This project follows the **Google Data Analytics** methodology by:
1.  **Engineering:** Managing 500+ potential data records via structured Room Entities.
2.  **Analysis:** Implementing real-time filtering logic to enhance user decision-making speed.
3.  **Humanization:** Designing a UI that reduces cognitive load, aligned with my goal of developing **Psychological Capital (PsyCap)** in digital environments.

## 📸 Wireframes & Design
The design phase involved creating detailed wireframes in **Figma** to map the user journey, ensuring the final Android UI adheres strictly to the planned architectural layout.
